# 💬 Sistema de Mensageria P2P (Peer-to-Peer)

Este projeto implementa um sistema simples de troca de mensagens utilizando o modelo **P2P (peer-to-peer)** com sockets UDP em Python.


## 🚀 Funcionalidades

- 📩 Envio de mensagens entre nós
- 📥 Recebimento de mensagens via socket UDP
- 🗂️ Histórico de conversas por contato
- 🔁 Encaminhamento de mensagens
- 🧵 Execução concorrente com threads


## 🛠️ Tecnologias utilizadas

- Python 3
- Socket (UDP)
- Threading
- JSON
